# Three‑Word List
This program asks the user to enter three words, splits them into separate items, and stores them in a list.

## Features
- Asks the user for three words
- Splits the input into individual words
- Stores the words in a list
- Prints the full list


## Learning Purpose
- A small exercise to practice:
- user input
- .split() for word separation
- list creation
- simple output
